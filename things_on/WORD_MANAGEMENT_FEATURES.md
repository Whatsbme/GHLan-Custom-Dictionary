## 🚀 **使用方法**

### **启动应用**
```bash
python run.py
```

### **导航到词条管理**
1. 打开应用
2. 点击左侧导航菜单
3. 选择"词条列表"查看所有词条
4. 选择"添加词条"创建新词条

### **词条操作**
- **查看详情**: 点击词条卡片
- **编辑词条**: 在详情页点击编辑按钮
- **删除词条**: 在详情页点击删除按钮
- **收藏词条**: 点击心形图标

### **搜索和筛选**
- 使用搜索栏进行关键词搜索
- 选择搜索类型（全部、字序号、拉丁写法等）
- 使用排序按钮选择排序方式
- 使用筛选按钮选择显示范围
