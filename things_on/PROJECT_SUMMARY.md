# GHLan自定义字典 - 项目总结

## 🎯 项目概述

GHLan自定义字典是一款功能完整的离线字典应用，采用传统水墨画风格设计，支持词条管理、高级搜索、导入导出、工具功能等。项目采用Python + Kivy + KivyMD技术栈，实现了现代化的用户界面和完整的业务功能。
